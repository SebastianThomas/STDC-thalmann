# Decompression algorithms for the STDC project

## Limitations and Liability, Safety Warning

This project is experimental software and has not been certified, validated, or 
approved for use in life-support, diving, decompression planning, or other 
safety-critical applications.

The algorithms implemented in this project are derived from publicly available 
research and concepts used in commercial dive computers, but may differ 
significantly in implementation, behavior, testing, and validation.

This software may contain errors or inaccuracies that could result in equipment 
failure, incorrect decompression calculations, serious injury, or death.
